## MODIFIED Requirements

### Requirement: The AI system prompt documents all supported node types and their properties
The system prompt SHALL list all supported node types including `math`, `functionGraph`, and `parametricGraph` alongside `group`, `rect`, and `text`. For each new node type, the prompt SHALL describe the required and optional properties, provide usage guidance, and include constraints (e.g., math nodes require valid LaTeX, graph expressions use mathjs syntax). The prompt SHALL instruct the model to use `math` nodes for equations and mathematical notation, `functionGraph` for y=f(x) curves, and `parametricGraph` for parametric curves.

#### Scenario: The system prompt includes math node documentation
- **WHEN** the system prompt is constructed
- **THEN** it contains guidance for the `math` node type including the `latex` property and valid LaTeX syntax

#### Scenario: The system prompt includes graph node documentation
- **WHEN** the system prompt is constructed
- **THEN** it contains guidance for `functionGraph` and `parametricGraph` node types including `fn`, `fnX`, `fnY`, range properties, and `drawProgress` animation

### Requirement: AI-generated video descriptions with new node types validate against the schema
Video descriptions generated by the AI that include `math`, `functionGraph`, or `parametricGraph` nodes SHALL pass `videoDescriptionSchema` validation without errors when the nodes include all required fields and valid property values.

#### Scenario: An AI-generated scene with a math node passes validation
- **WHEN** the AI generates a video description containing a `math` node with valid `latex`, `fontSize`, `width`, and `height`
- **THEN** `videoDescriptionSchema.safeParse` succeeds

#### Scenario: An AI-generated scene with a functionGraph node passes validation
- **WHEN** the AI generates a video description containing a `functionGraph` node with valid `fn`, `xRange`, `yRange`, `width`, `height`, and an `animate` block for `drawProgress`
- **THEN** `videoDescriptionSchema.safeParse` succeeds
